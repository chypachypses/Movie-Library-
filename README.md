**Автор:** Саитбаталов Никита 
**Вариант:** Movie Library   
**Дата сдачи:** апрель 2026

---

## Описание программы

*"Movie Library — это графическое приложение для ведения личной коллекции фильмов. Позволяет добавлять записи о фильмах (название, жанр, год выпуска, рейтинг), удалять их, фильтровать по жанру и году, а также просматривать статистику коллекции. Все данные автоматически сохраняются в JSON-файл."*

---

## Требования для запуска

Что нужно установить на компьютер:

- Python 3.7 или выше
- Tkinter (обычно уже установлен с Python)

Дополнительные библиотеки не требуются — используется только стандартная библиотека Python.

## Как пользоваться
1.	При запуске открывается графическое окно с полями для ввода.
2.	Заполните поля: Название, Жанр, Год выпуска, Рейтинг.
3.	Нажмите кнопку «Добавить фильм» — запись появится в таблице.
4.	Для фильтрации введите текст в поля «Жанр» или «Год» в секции фильтрации.
5.	Для удаления выделите фильм в таблице и нажмите «Удалить выбранный фильм».
6.	Данные автоматически сохраняются в movies.json.

## Архитектура проекта (для ООП-вариантов)
movie-library/
├── movie_library.py      # Точка входа + весь код приложения
├── movies.json           # Файл с сохранёнными данными (создаётся автоматически)
├── .gitignore            # Игнорируемые файлы для Git
└── README.md             # Этот файл

Структура классов:
MovieLibrary (главный класс)
├── Movie (вложенный класс для данных)
│   ├── to_dict()      # преобразование в JSON
│   └── from_dict()    # загрузка из JSON
├── create_widgets()   # создание интерфейса
├── add_movie()        # добавление фильма
├── delete_movie()     # удаление фильма
├── filter_movies()    # фильтрация списка
├── validate_year()    # проверка года
├── validate_rating()  # проверка рейтинга
├── save_movies()      # сохранение в JSON
├── load_movies()      # загрузка из JSON
└── update_stats()     # обновление статистики

## Тестирование
Тип теста	Пример	Результат
Позитивный	Добавление фильма Титаник (1997, 7.9)	✅ Фильм добавлен
Позитивный	Фильтр по жанру «Драма»	✅ Показаны только драмы
Позитивный	Фильтр по году «1994»	✅ Показаны фильмы 1994 года
Позитивный	Удаление выбранного фильма	✅ Фильм удалён
Негативный	Ввод пустого названия	❌ Ошибка: название не может быть пустым
Негативный	Ввод года «абв» вместо числа	❌ Ошибка: год должен быть целым числом
Негативный	Ввод года «1800»	❌ Ошибка: год должен быть не меньше 1888
Негативный	Ввод года «2026»	❌ Ошибка: год не может быть больше текущего
Негативный	Ввод рейтинга «15»	❌ Ошибка: рейтинг должен быть от 0 до 10
Негативный	Ввод рейтинга «-3»	❌ Ошибка: рейтинг должен быть от 0 до 10
Негативный	Ввод рейтинга «8,5» (с запятой)	❌ Ошибка: рейтинг должен быть числом
Негативный	Удаление без выбора фильма	⚠️ Предупреждение: выберите фильм
Граничный	Год = 1888 (минимальный)	✅ Принято
Граничный	Год = 2026 (текущий)	✅ Принято
Граничный	Рейтинг = 0.0 (минимальный)	✅ Принято
Граничный	Рейтинг = 10.0 (максимальный)	✅ Принято
Граничный	Название из 100 символов	✅ Принято
