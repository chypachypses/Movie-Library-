# Личная кинотека (Movie Library)

**Автор:** Саитбаталов Никита

## Описание
GUI-приложение на Python (Tkinter) для учета просмотренных фильмов. Позволяет добавлять фильмы, фильтровать список по жанру и году выпуска. Данные сохраняются в формате JSON.

## Функционал
1.  **Добавление:** Название, жанр, год, рейтинг.
2.  **Валидация:** Проверка на пустые поля, числовой год, рейтинг от 0 до 10.
3.  **Фильтрация:** Поиск по жанру (частичное совпадение) и точному году.
4.  **Сохранение:** Автоматическое сохранение в `movies.json`.

## Как использовать
1.  Запустите приложение: `python main.py`
2.  Заполните поля в верхней части и нажмите "Добавить фильм".
3.  Используйте панель фильтрации, чтобы отсеять фильмы.
4.  Нажмите "Сброс", чтобы увидеть весь список.
